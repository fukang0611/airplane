# Skybreaker

一款使用原生 HTML、CSS 和 JavaScript 实现的竖屏飞机射击游戏。游戏基于 Canvas 绘制，支持桌面端和移动端，无需安装依赖或构建即可运行。

## 运行

直接使用浏览器打开根目录下的 `index.html`。

也可以在项目目录启动本地静态服务器：

```bash
python3 -m http.server 4173
```

然后访问 `http://127.0.0.1:4173/`。

## 操作

- 方向键或 `WASD`：移动战机
- 鼠标或触摸拖动：战机按滑动方向和距离同步移动，按下时不会跳向触点
- `Space`：使用特殊弹药（未开始时用于出击）
- `P` 或 `Esc`：暂停/继续
- `C`：在已营救的僚机之间切换；移动端可点击僚机状态区
- `Enter`：开始或重新开始
- `1` / `2`：关卡结束时选择航线

战机会自动射击。敌弹贴近战机但没有命中时会触发擦弹并积累能量，充满后自动进入短时间 Overdrive，提升射速、伤害和得分倍率。

## 战役与成长

- 四种主题空域分别包含滑流、雷击、热区和引力井机制。
- 每关拥有护送、追击、破坏或逃脱支线任务，完成任务会收集异常信号。
- Boss 拥有可单独破坏的炮塔、护盾或环绕节点，并会随部件损毁进入新的攻击阶段。
- 关卡结束后可以选择安全或高风险航线；航线会改变下一关压力、得分倍率和战术模块。
- 战术模块包含护盾、擦弹反应堆、自动修复、武器加速、拾取磁吸和相位闪避。
- 完成第四关后进入强化周目，敌方生命、弹速、环境频率和精英编队继续升级。
- 连续完成支线任务会稳定异常信号，并可能打开额外战斗。
- 敌军编队包含护盾、维修、指挥、干扰、隐形和突击单位；支援范围与连线会提示需要优先处理的战术节点。
- 战役后段加入峡谷追逐与轨道电梯攀升：航道会收缩，卷轴速度会变化，需要摧毁地面电池或钳制节点，并避开可破坏障碍。

## 僚机

击破特定支援单位会释放救援舱。营救驾驶员后可以部署并切换僚机：`ROOK` 拦截接近玩家的敌弹，`LANCER` 自动发射追踪导弹，`MENDER` 周期性维修装甲。

## 武器与拾取

拾取不同类型的武器会切换武器，但保留当前武器等级；拾取同类型武器或蓝色 `U` 升级道具会提升全局武器等级，最高为三级。击落敌机还可能获得特殊弹药或装甲补给。

军械轮盘会显示当前武器。持续射击轮盘会依次切换候选武器，停止射击后靠近即可锁定并拾取；若锁定的类型与当前武器相同，则按相同规则提升全局武器等级。

除原有武器外，电弧武器会在邻近敌机间跳跃，重力炮会牵引敌机并捕获敌弹，回旋刃会在返航时再次穿过敌军。

后续玩法安排和不可回归规则记录在 [ROADMAP.md](./ROADMAP.md)。
