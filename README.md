# Skybreaker

一款使用原生 HTML、CSS 和 JavaScript 实现的竖屏飞机射击游戏。游戏基于 Canvas 绘制，支持桌面端和移动端，无需安装依赖或构建即可运行。

## 运行

直接使用浏览器打开根目录下的 `index.html`。

## 操作

- 方向键或 `WASD`：移动战机
- 鼠标或触摸拖动：移动战机
- `Space`：使用特殊弹药（未开始时用于出击）
- `P` 或 `Esc`：暂停/继续
- `Enter`：开始或重新开始

战机会自动射击。击落敌机可累计分数和连击，并有机会获得武器、特殊弹药或装甲补给。

## 项目结构

```text
.
├── index.html       # 页面入口与游戏界面
└── src/
    ├── game.js      # 游戏状态、战斗逻辑、绘制、输入和音效
    └── styles.css   # 页面布局、HUD 与响应式样式
```
